# Invoice Management System (Full Stack MERN)

A full-stack Invoice Details Module built using the MERN stack.

## 🚀 Features

- Create Invoice
- Edit Invoice
- Archive / Restore Invoice
- Add Payments
- Prevent Overpayment
- Auto Status Update (DRAFT → PAID)
- Professional PDF Generation
- Authentication (Login / Signup)

---

## 🗄️ Tech Stack

Frontend:
- React (Vite)
- Axios
- React Router
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- PDFKit (for PDF generation)
- JWT Authentication
